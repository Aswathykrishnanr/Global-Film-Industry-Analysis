# 🎬 Global Film Industry Analysis — Power BI Dashboard

An interactive Power BI dashboard analyzing global cinema trends using the IMDB movies dataset.

## 📊 Dashboard Overview

This dashboard explores the global film industry through data — uncovering patterns in box office performance, genre trends, ratings and regional cinema insights.

## 🔍 Key Insights

- Top performing genres by revenue and ratings
- Global box office trends over the years
- IMDB rating distributions across countries and genres
- Most successful directors and production companies
- Regional cinema performance comparison

## 🛠️ Tools Used

- **Power BI** — Dashboard and visualizations
- **Dataset** — IMDB Movies Dataset (CSV)
- **Data Cleaning** — Power Query inside Power BI
